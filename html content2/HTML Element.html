Main Root

This tag referred to the main or root(starting tag) element, which other elements are decendent to this element tag.

  • html

    • Document Metadata

    This element tag are containers for the data or information (Metadata), style for the content of webpage.

  • base

    • "base" tag element indentify or state the base "url" or "uri" link for all relative content of document. This tag should be inside the "head " tag and don't have a end tag.

  • head

    • "head tag" is used for the container of head element. "head" elements contain important information and child element such as title, style, base , link, meta, nonscript, script \

  • link

    • This element use to establish a relationship or link between document and external resources, and external style .

  • meta

    • meta is used to provide a information that can't represent by other elements. Information that written inside meta will not be display but, browser will utilised this element to know how to display the content or control the webpage or document.

  • style

    • style element contain the style information of the document or it manage the design, look and how those the element inside behave.

  • title

    • Refers to the title that display on the browser.
